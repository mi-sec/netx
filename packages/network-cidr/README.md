# `@mi-sec/network-cidr`

network cidr utility for parsing, expanding, and validating IPv4 CIDR blocks

- Cross-platform - Support for:
    - CommonJS, ECMAScript Modules and UMD builds
    - Node >= 13.2.x (8, 10, 12 in progress)
